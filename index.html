Instructions: pick 3 initial offsets using the arrows (remember them), there are 26 possible positions for each rotor.
Every change in the initial settings of the machine will result in a different coding being used.
When you press a letter on the keyboard its coded counterpart will light up on the screen, write it down.
Now after you finished writing, if you want to decode your message set the rotors to their initial values and type the text you want to decode.
